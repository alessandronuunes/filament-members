# Changelog

All notable changes to `filament-member` will be documented in this file.

## 1.0.0 - 2026-01-31

### Added
- Initial release
- Member management interface
- Invitation system (email and shareable links)
- Role-based access control (Owner, Admin, Member)
- Multi-language support (English, Portuguese BR)
- Email notifications for invitations
- Automatic invitation acceptance after login
- Configurable models, routes, and behavior
- Responsive UI with Filament components
- Validation to prevent duplicate invitations

### Features
- Up to 5 email invitations at once
- Pending invitations management
- Member role management
- Configurable invitation expiration
- Generic invitation links (no email required)
- Custom role enum support
